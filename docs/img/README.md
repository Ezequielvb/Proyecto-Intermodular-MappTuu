## Cómo generarlas

1. Abrir https://vercel-angular-mapp-tuu.vercel.app/ (y app Android si aplica).
2. Capturar pantalla (misma resolución recomendada, p. ej. 1440×900).
3. Guardar en esta carpeta y referenciar en el `README.md` principal:

```markdown
![Listado de actividades](docs/img/02-listado-actividades.png)
```

**No subir** capturas con datos personales reales de terceros sin consentimiento.
